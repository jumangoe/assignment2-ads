## Project summary
In this project, I implemented a simple banking system using Java data structures. I used LinkedList to store bank accounts, Queue to manage account requests and bill payments, Stack to track transaction history, and Array for fixed data storage. This project helped me understand the difference between logical and physical data structures and how they are applied in real-world scenarios.
